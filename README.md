# 📚 Trabajo Práctico Obligatorio — Programación II (2025)

## 🎯 Objetivo

Implementar nuevos TDA y resolver problemas utilizando exclusivamente las estructuras permitidas por la materia.

---

## 🚫 Restricciones Obligatorias

- No usar Collections Framework.
- No usar ArrayList.
- No usar HashMap.
- No usar LinkedList de Java.
- No usar estructuras ajenas a la cursada.

### Estructuras permitidas

- Arreglos.
- Listas enlazadas vistas en clase.
- Objetos propios.
- Los ocho TDA de la materia.
- P2lib.jar.
